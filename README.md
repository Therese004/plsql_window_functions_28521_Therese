### STEP ONE: PROBLEM DEFINITION
## UNIVERSITY ENROLLMENT DATA ANALYSIS
# BUSINESS CONTEXT
THIS WORK FOCUSES ON A UNIVERSITY ACADEMIC ENVIRONMENT WHERE THE ACADEMIC DEPARTMENT MANAGES STUDENT RECORDS,COURSE INFORMATION, AND ENROLLMENT DATA FOR EACH SEMESTER.
# DATA CHALLENGE
STUDENT INFORATION, COURSE DETAILS,AND ENROLLMENT RECORDS ARE STORED IN SEPARATE TABLES.THIS MAKES IT DIFFICULT TO KNOW THE TOP PERFORMING STUDENTS,COMPARE RESULTS FOR EACH SEMESTER AND ANALYZE COURSE ENROLLMENT TRENDS
# EXPECTED OUTCOME
FOR DOING ANALYSIS IS EXPECTED TO IDENTIFY TOP STUDENTS, EVALUATE COURSE ENROLLMENT PATTERNS, AND SUPPORT BETTER PERFORMANCE EVALUATION.
### STEP TWO: SUCCESS CRITERIA
     1. IDENTIFY TOP_PERFORMING STUDENTS USING RANK() ACCORDING TO THEIR SCORES.
     2. CALCULATE TOTAL SCORES OR CREDITS PER STUDENT USING SUM() OVER().
     3. COMPARE STUDENT PERFORMANCE BETWEEN SEMESTER USING LAG()
     4.DIVIDE STUDENTS INTO PERFORMANCE GROUPS USING NTILE(4).
     5. ANALYZE PERFORMANCE TRENDS USING MOVING AVERAGES WITH AVG() OVER().
